## 📚 Sobre o Curso

Este repositório contém todas as atividades, projetos e exercícios desenvolvidos durante as aulas de Desenvolvimento Web. Aqui você encontrará desde os primeiros passos com HTML até projetos mais complexos envolvendo CSS, formulários e muito mais!

### 🎯 Objetivos do Curso
- Aprender os fundamentos do **HTML5** e **CSS3**
- Desenvolver páginas web responsivas e modernas
- Criar formulários interativos
- Construir um portfólio profissional
- Aplicar boas práticas de desenvolvimento web
